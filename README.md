# IT Help Desk Ticketing Simulation Lab

## Overview

This project simulates a real-world Tier 1 IT Support environment using a ticketing system (Peppermint).  

The goal of this lab is to demonstrate structured troubleshooting, proper documentation, escalation logic, and fundamental IT support knowledge.

All scenarios are based on common Service Desk incidents encountered in corporate environments.



## Skills Demonstrated

- Incident documentation and ticket management
- Structured troubleshooting methodology
- Layer-based network diagnostics (OSI understanding)
- DHCP & DNS troubleshooting
- Active Directory user management concepts
- Microsoft Outlook troubleshooting
- Printer and hardware issue diagnostics
- Escalation procedures to Tier 2 teams
- Professional written communication



## Environment

- Windows 10/11 Workstation
- Active Directory (Lab)
- DHCP & DNS simulated environment
- Ticketing System (Peppermint – Docker-based)
